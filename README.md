## 🍔 Swiggy Sales Analytics Dashboard | Excel  

### 🔥 Overview  
This project analyzes Swiggy sales data using Excel to uncover insights on **sales trends, customer behavior, and regional performance** through an interactive dashboard.

---

### 🛠️ Tools & Techniques Used  
- Microsoft Excel  
- Pivot Tables  
- Pivot Charts  
- Data Cleaning & Transformation  
- Dashboard Design  

---

### ⚙️ What I Implemented  
- Created KPI metrics (Sales, Orders, Ratings)  
- Built Monthly, Weekly & Daily trend analysis  
- Top 5 cities by sales  
- Veg vs Non-Veg analysis (Donut chart)  
- Quarterly performance table  
- Map visualization (Sales by State)  
- Added filters (Month, State, Restaurant)  
- Navigation buttons (Dashboard / Data / Analysis)  
- Financial Year calculation using formula  

---

### 📊 Dashboard Features  
- Interactive filters  
- KPI cards  
- Trend analysis charts  
- Map-based insights  
- Clean and user-friendly layout  

---

### 🚀 Outcome  
This dashboard helps in understanding **sales performance, customer preferences, and regional trends**, enabling better decision-making.
